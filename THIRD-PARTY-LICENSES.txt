### Technical Implementation
This application does NOT modify, link to, or incorporate 7-Zip source code. Instead, it:
- Bundles the unmodified 7-Zip executable for user convenience
- Communicates with 7-Zip via subprocess calls and command-line interface
- Maintains complete process separation between application code and 7-Zip
- Functions as a CLI wrapper that calls 7-Zip as an external tool

This approach is similar to how Git GUI clients bundle the Git executable, or how video editors bundle FFmpeg - the LGPL component remains separate and unmodified.
